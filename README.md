## 📌 주요 기능
#### ABOUT
 - 저의 학력과 이메일과 같은 저에 대하여 간단히 알 수 있는 정보를 적어 놓았습니다.
#### SKILLS
 - 제가 사용할 수 있는 언어를 그래프 형식으로 보기 쉽게 정리했습니다.
#### PROJECT
 - 여태까지 만들어온 프로젝트를 정리했습니다. 프로젝트의 간단한 설명과 기간 등이 정리되어 있습니다.
#### AWARDS
 - 수상내역을 정리했습니다.
